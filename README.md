## 🌌 Project Vision
**SMARTSANIX X1** is a next-generation IoT ecosystem engineered to transform public sanitation. By integrating real-time air quality analysis, occupancy tracking, and automated resource management, it ensures high hygiene standards in high-traffic hubs like airports, hospitals, and smart cities.

> [!IMPORTANT]
> **Mission:** To eliminate sanitation gaps through data-driven automation and real-time accountability.

---

## 🚀 Key Features

| Feature | Tech Stack | Impact |
| :--- | :--- | :--- |
| **Occupancy Detection** | PIR Motion Sensors | Optimizes cleaning schedules based on traffic. |
| **Odor Monitoring** | MQ135 Gas Sensor | Triggers ventilation or cleaning alerts instantly. |
| **Water Monitoring** | Ultrasonic/Flow Sensors | Prevents dry-tap scenarios and water wastage. |
| **Live Dashboard** | JS / CSS Grid | Futuristic UI for facility managers. |
| **Predictive Alerts** | Logic-based Triggers | Reduces downtime and improves user experience. |
